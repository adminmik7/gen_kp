# Web-Form КП

Веб-приложение для создания коммерческих предложений (КП) с возможностью авторизации, управления пользователями, генерации документов и печати.

## Возможности

- **Создание КП** — заполнение формы с информацией о заказчике, условиях поставки и товарах
- **Управление товарами** — добавление, редактирование, удаление позиций с автоматическим расчётом НДС и итоговой суммы
- **Drag-n-Drop** — перетаскивание строк для изменения порядка товаров
- **Сумма прописью** — автоматическое преписание итоговой суммы текстом
- **Печать / PDF** — оптимизированная версия для печати на A4
- **Сохранение / Загрузка** — экспорт и импорт данных в формате JSON
- **Логотип и факсимиле** — загрузка изображений для оформления документа
- **Авторизация** — система входа с хранением сессии в localStorage
- **Управление пользователями** — админ-панель для добавления, редактирования и удаления пользователей

## Структура проекта

| Файл | Описание |
|------|----------|
| `index.html` | Основное приложение — форма КП, товары, печать |
| `aut.html` | Панель управления пользователями (только для администраторов) |
| `db.json` | База пользователей (логин, пароль, роль, контакты) |
| `server.js` | Express-сервер на порту 8000 с API для пользователей |

## Установка и запуск

### Требования
- Node.js
- npm

### Установка зависимостей
```bash
npm install express
```

### Запуск сервера
```bash
node server.js
```

Сервер запустится на `http://localhost:8000`

|-------|--------|------|
| admin | 8675309 | admin |
| manager | manager | user |

## Использование

1. Откройте `http://localhost:8000` в браузере
2. Войдите в систему (логин / пароль из таблицы выше)
3. Заполните информацию о заказчике и условиях поставки
4. Добавьте товары в таблицу
5. Нажмите "Печать" для печати или "Сохранить" для экспорта в JSON
6. Для управления пользователями нажмите кнопку 👤 в правом верхнем углу (только для администраторов)

## Технологии

- Чистый HTML / CSS / JavaScript (без фреймворков)
- Express.js (Node.js) для серверной части
- SHA-256 (чистая реализация на JS) для хеширования паролей
- html2canvas / jsPDF (подключены, PDF временно недоступен)
- localStorage для хранения сессии
